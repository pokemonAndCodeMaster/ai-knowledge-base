# 统一知识系统维护法则 (Schema)

## 📌 概述
此文件是 `ai-librarian` 的运行准则。整个知识库分为：
- `raw/` : 只读原始资料。
- `wiki/` : 二次处理后的网状知识卡片。

## 📜 规则
1. **纯中文维护**：哪怕源头文档是全英文，产出的 Wiki 文件、日志、及索引记录必须是中文格式。
2. **知识提取优先级**：优先从原始文档中析出“心智模型”与“可实操的步骤框架”（比如 MIT 高效学习的 48小时原则，Karpathy 的 Zettelkasten 管理）。
3. **强制关联**：每新建一个概念，都要反身思考：这与目前库里已有的知识有什么关联或冲突？并打上 `[[标签]]` 记号。
4. **统一更新点**：
   - 目录：增加页面后必须更新 `index.md`。
   - 动作：增加动作后必须在 `log.md` 留档。
