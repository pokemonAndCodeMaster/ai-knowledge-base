# Almirant

## 简介
Almirant 被定义为面向“人类-智能体团队”的操作系统。其核心理念是将“思考”与“执行”分离，让开发者从持续的 Prompt、等待和 Code Review 的瓶颈中解脱出来。

## 核心功能
1. **思考与执行分离**：人类负责决策和规划，智能体负责执行。
2. **并行执行仪表盘**：可以同时监控多个智能体在不同任务上的进展，解决“Babysitting (保姆式监控)”的问题。
3. **持久化上下文 (Persistent Context)**：解决跨 Session 的遗忘问题，保证智能体能记住之前的架构决策和项目模式。
4. **共享记忆**：团队成员和不同智能体之间共享任务背景。
5. **结构化任务生命周期**：计划 (Plan) → 实施 (Implement) → 评审 (Review) → 部署 (Deploy)。

## 适用场景
- **Solo Builder**：一人公司，需要多个智能体并行工作。
- **自由职业者**：频繁切换项目上下文。
- **小型团队/机构**：需要透明的智能体协作仪表盘。
